# tag list

untuk membuat list pada html terdapat 3 tag tiap tag fungsinya berbeda

  • tag ol
  • untuk membuat list terurut
  • tag ul
  • untuk membuat list tidak terurut>
  • tag dl
  • untuk membuat list berpasangan
  • tag dt berpasangan tag dd

    • catatan : didalam tiap tag harus ada tag li

    A. tag ol

    pertama ada tag ol untuk membuat list terurut yang di dalamnya terdapat tag li

  • untuk mengubah urutannya menjadi a, A, 1, I dapat menambahkan elemen:type
  • type
  • a
  • A
  • 1
  • I

    • contoh

    1. bangun pagi
    2. sarapan
    3. tidur pagi

    2. sarapan
    3. tidur pagi

      type=a

    1. bangun pagi
    2. sarapan
    3. tidur pagi

      type=I

    1. bangun pagi
    2. sarapan
    3. tidur pagi

      type=i

    1. bangun pagi
    2. sarapan
    3. tidur pagi

    #tag hyperlink

    • untuk membuat link harus di buka dengan tag= a. di dalama tag= a terdapat elemen=href, value=untuk mengisi alamat yg dituju
    • untuk membuka tab baru di browser ketika di klik harus menggunakan elemen=target, value=_blank.
    • untuk mengarah langsung ke halaman tertentu yang terdapat di dalam link kita harus menambahkan #kemudian sesuaikan id target yang dituju

    #tag form

    form adalah sebuah tag pada html yang digunakan untuk mengelolah data dari input yang dilakukan oleh user baik itu saat user mengisih data pada kolom atau memilih sesuatu pada komboboks seperti ceklis

    cara membuat form

    yang pertama kita membutuhkan tag=form, sebelum kita membuat elemen atau tag lainnya kita bungkus dulu nmenggunakan tag=form. ini di lakukan untuk mengetahui bahwa satu kumpulan adalah bagian dari satu form

    beberapa tag yang terdapat di dalam form yang dapat atau sering di gunakan

  • input
  • untuk membuat teksfilt, radiobutton, komboboks dan lain-lain
  • input dimana user bisa menginputkan sesuatu
  • textarea
  • mirip seperti textfilt tapi lebih banyak yang bisa kita isikan seperti komentar pesan dll
  • select
  • digunakan untuk membuat dropdown atau komboboks
  • button
  • sebuah tombol yang di gunakan untuk mengirimkan datanya
  • label
  • untuk menandai sebuah elemen input

    • A. input

    beberapa elemen yang terdapat di dalam imput

  • elemen: type = ...
  • value=text
  • value=password
  • value=radiobutton
  • value=checkbox

    • conto tag input di dalam form

    1. tag=input di dalam form biasanya dibuatkan tag=label di atasnya untuk memberitahu html bahwa saya akan membuat label

      2. tag=label di dalamnya terdapat elemen dan value yaitu elemen= for yang valuenya harus sama dengan value id yang terdapat di tag=input agar terhubung satu sama lain. ini di lakukan ketika menambahkan keterangan atau nama

    • type = text

    • type = password

    • type = radiobuttoon

    radiobutton ini harus di tambahkan elemen dan value lagi ketika lebih dari satu dan ingin memilih salah satunya .elemennya yaitu =name dan isi value nya harus sama jika lebih dari satu

    • type = checkbox

    chackbox hampir mirip dengan radiobutton chackbox dapat memilih beberapa pilihan tidak seperti radiobutton harus memilih salah satunya

      untuk membuat tombol

    • type = submit,button dan riset

    untuk membuat tombol pada imput untuk mengirim datanya dapat menggunaka submit,riset dan button tpi ini tidak terlalu di sarankan

    jika ingin mengganti tulisan pada submit kita dapat memberikan elemen:value=nama yang di inginkan...

    tag yang di sarankan ketika mau membuat tombol yaitu tag=button

    B. textarea

    textarea untuk membuat tempat penulisan yang panjang buat user seperti menulis komentar atau pesan dengan cara membuat tag=textarea di dalam tag=form

    C. select

    select untuk membuat pilihan yang sudah di sediakan untuk user yang perjejeran ke bawang sperti membuka jendela baru dan di dalamnya terdapat banyak pilihan

    untuk membuat nya kita harus membuta tag=select di dalam tag=form, kemudian di dalam tag select terdapat lagi tag=option
